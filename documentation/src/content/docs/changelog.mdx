## How Releases Work

`small-rules` uses [bumpp](https://github.com/antfu/bumpp) for versioning. Releases are triggered by pushing a `v*.*.*` tag to the repository.

Each release is published to NPM via Trusted Publishing (OIDC) and a GitHub Release is created automatically with `changelogithub`.
