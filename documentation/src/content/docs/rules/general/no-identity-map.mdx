`small-rules/no-identity-map` reports code described by its source metadata: `disallow pointless identity '.map()' calls that return the parameter unchanged`. It reports with the message `Pointless identity '.map()' call on Array. Use 'table.clone(array)' or '[...array]' instead.`.

## Rule Details

This rule is configured as `small-rules/no-identity-map` and reports the concrete syntax covered by its implementation in `src/rules/no-identity-map.ts`.

<Aside type="note">

Rule messages:

```text
identityArrayMap: Pointless identity `.map()` call on Array. Use `table.clone(array)` or `[...array]` instead.
identityBindingMap: Pointless identity `.map()` call on Binding. Use the original binding directly.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-identity-map": [
			"error",
			{
				"bindingPatterns": ["binding"]
			}
		]
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
scaleBinding.map((v) => v);
```

</RuleExample>

When auto-fixed, this becomes:

```ts
scaleBinding;
```

<RuleExample type="pass" title="Passes">

```ts
binding.map((v) => v + 1);
```

</RuleExample>
