## Rule Details

This rule is configured as `small-rules/no-unused-imports` and reports the concrete syntax covered by its implementation in `src/rules/no-unused-imports.ts`.

## Configuration

<RuleOptions rule="no-unused-imports" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
import { used, unused } from "./module";

used();
```

</RuleExample>

When auto-fixed, this becomes:

```ts
import { used } from "./module";

used();
```

<RuleExample type="pass" title="Passes">

```ts
import { used } from "./module";

used();
```

</RuleExample>
