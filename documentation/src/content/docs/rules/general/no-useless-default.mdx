`small-rules/no-useless-default` reports code described by its source metadata: `scaffold for detecting useless default values across 236 default-property classes`. It reports with the message `Remove "{{propertyName}}" from {{className}}. It already matches the default value.`.

## Rule Details

This rule is configured as `small-rules/no-useless-default` and reports the concrete syntax covered by its implementation in `src/rules/no-useless-default.ts`.

<Aside type="note">

Rule message:

```text
uselessDefault: Remove "{{propertyName}}" from {{className}}. It already matches the default value.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-useless-default": "error"
	}
}
```
