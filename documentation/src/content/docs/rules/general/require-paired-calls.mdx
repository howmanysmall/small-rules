## Examples

<RuleExample type="fail" title="Fails">

```ts
function test() {
	setup();
	doWork();
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function test() {
	setup();
	doWork();
	cleanup();
}
```

</RuleExample>
