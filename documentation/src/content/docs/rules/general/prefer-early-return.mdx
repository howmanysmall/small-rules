## Examples

<RuleExample type="fail" title="Fails">

```ts
function foo() {
	if (something) {
		doSomething();
		doSomethingElse();
	}
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function foo() {
	if (!something) {
		return;
	}
	doSomething();
	doSomethingElse();
}
```

</RuleExample>
