`small-rules/no-error` reports code described by its source metadata: `use throw instead of error()`. It reports with the message `Replace error(...) with throw.`.

## Rule Details

This rule is configured as `small-rules/no-error` and reports the concrete syntax covered by its implementation in `src/rules/no-error.ts`.

<Aside type="note">

Rule message:

```text
noError: Replace error(...) with throw.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-error": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
error("Hello");
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
Log.error("Hello");
```

</RuleExample>
