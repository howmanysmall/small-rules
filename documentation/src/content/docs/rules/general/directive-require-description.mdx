`small-rules/directive-require-description` reports code described by its source metadata: `require description for 'eslint-disable' and 'eslint-enable' directives`. It reports with the message `Missing description for '{{kind}}' comment.`.

## Rule Details

This rule is configured as `small-rules/directive-require-description` and reports the concrete syntax covered by its implementation in `src/rules/directive-require-description.ts`.

<Aside type="note">

Rule message:

```text
missingDescription: Missing description for '{{kind}}' comment.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/directive-require-description": ["error", {}]
	}
}
```
