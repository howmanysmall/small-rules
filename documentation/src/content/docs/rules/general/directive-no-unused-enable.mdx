`small-rules/directive-no-unused-enable` reports code described by its source metadata: `disallow unused 'eslint-enable' comments`. It reports with the message `Unused '{{kind}}' comment. No reported rules are disabled.`.

## Rule Details

This rule is configured as `small-rules/directive-no-unused-enable` and reports the concrete syntax covered by its implementation in `src/rules/directive-no-unused-enable.ts`.

<Aside type="note">

Rule messages:

```text
unused: Unused '{{kind}}' comment. No reported rules are disabled.
unusedRule: '{{ruleId}}' rule is disabled but never reported. Please remove unnecessary '{{kind}}' comment.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/directive-no-unused-enable": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
/* oxlint-enable no-console */
const x = 1;
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
/* oxlint-disable no-console */
const x = 1;
/* oxlint-enable no-console */
```

</RuleExample>
