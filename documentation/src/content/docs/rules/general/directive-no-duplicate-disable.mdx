`small-rules/directive-no-duplicate-disable` reports code described by its source metadata: `disallow duplicate 'eslint-disable' comments`. It reports with the message `ESLint rules have been disabled already.`.

## Rule Details

This rule is configured as `small-rules/directive-no-duplicate-disable` and reports the concrete syntax covered by its implementation in `src/rules/directive-no-duplicate-disable.ts`.

<Aside type="note">

Rule messages:

```text
duplicate: ESLint rules have been disabled already.
duplicateRule: '{{ruleId}}' rule has been disabled already.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/directive-no-duplicate-disable": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
/* oxlint-disable no-console */
const x = 1;
/* oxlint-disable no-console */
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
/* oxlint-disable no-console */
const x = 1;
/* oxlint-enable no-console */
```

</RuleExample>
