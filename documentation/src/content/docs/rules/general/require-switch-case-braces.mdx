`small-rules/require-switch-case-braces` reports code described by its source metadata: `require braces around switch case bodies that span multiple lines`. It reports with the message `Wrap this switch case body in braces.`.

## Rule Details

This rule is configured as `small-rules/require-switch-case-braces` and reports the concrete syntax covered by its implementation in `src/rules/require-switch-case-braces.ts`.

<Aside type="note">

Rule message:

```text
wrapCaseBody: Wrap this switch case body in braces.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/require-switch-case-braces": [
			"error",
			{
				"metric": "lines"
			}
		]
	}
}
```
