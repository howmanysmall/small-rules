## Rule Details

This rule is configured as `small-rules/prefer-class-properties` and reports the concrete syntax covered by its implementation in `src/rules/prefer-class-properties.ts`.

## Configuration

<RuleOptions rule="prefer-class-properties" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
class Foo {
	constructor() {
		this.foo = "bar";
	}
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
class Foo {
	foo = "bar";
}
```

</RuleExample>
