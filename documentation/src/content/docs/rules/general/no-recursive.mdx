## Rule Details

This rule is configured as `small-rules/no-recursive` and reports the concrete syntax covered by its implementation in `src/rules/no-recursive.ts`.

Recursion is banned per the JPL Power of 10 rules. Use iteration — a loop or explicit stack — instead. The rule detects direct and indirect (mutual) recursion across named function declarations, function expressions, class methods, and variable-declared arrow/function expressions.

## Examples

<RuleExample type="fail" title="Fails">

```ts
function factorial(n: number): number {
	return n <= 1 ? 1 : n * factorial(n - 1);
}
```

```ts
function a(): void {
	b();
}
function b(): void {
	a();
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function factorial(n: number): number {
	let result = 1;
	for (let i = 2; i <= n; i++) result *= i;
	return result;
}
```

</RuleExample>
