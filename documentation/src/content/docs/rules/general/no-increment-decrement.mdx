## Rule Details

This rule is configured as `small-rules/no-increment-decrement` and reports the concrete syntax covered by its implementation in `src/rules/no-increment-decrement.ts`.

## Configuration

<RuleOptions rule="no-increment-decrement" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
size++;
```

</RuleExample>

With `"allowAutofix": true`, this becomes:

```ts
size += 1;
```

<RuleExample type="pass" title="Passes">

```ts
array[size++] = value;
```

</RuleExample>
