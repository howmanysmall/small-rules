`small-rules/no-underscore-react-props` reports JSX attributes whose name begins with an underscore. Its source metadata describes it as: `Ban React property names that begin with an underscore in JSX`.

## Rule Details

This rule is configured as `small-rules/no-underscore-react-props` and reports the concrete syntax covered by its implementation in `src/rules/no-underscore-react-properties.ts`.

<Aside type="note">

Rule message:

```text
React prop '{{propName}}' starts with '_'. Remove the leading underscore from the prop name.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-underscore-react-props": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```tsx
<Component _foo={bar} />;
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
<Component foo={bar} />;
```

</RuleExample>
