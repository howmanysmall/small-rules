## Rule Details

This rule is configured as `small-rules/require-react-display-names` and reports the concrete syntax covered by its implementation in `src/rules/require-react-display-names.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```tsx
import { memo } from "@rbxts/react";

function ComponentNoMemo() {
    return <div />;
}

export default memo(ComponentNoMemo);
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
import { memo } from "@rbxts/react";

function ChecklistClaimButtonNoMemo() {
    return <div />;
}

export const ChecklistClaimButton = memo(ChecklistClaimButtonNoMemo);
ChecklistClaimButton.displayName = "ChecklistClaimButton";
export default ChecklistClaimButton;
```

</RuleExample>
