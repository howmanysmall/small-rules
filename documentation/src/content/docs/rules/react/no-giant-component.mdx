`small-rules/no-giant-component` reports code described by its source metadata: `report React components whose bodies exceed the configured size threshold`. It reports with the message `Component "{{name}}" is {{lineCount}} lines — consider breaking it into smaller focused components`.

## Rule Details

This rule is configured as `small-rules/no-giant-component` and reports the concrete syntax covered by its implementation in `src/rules/no-giant-component.ts`.

<Aside type="note">

Rule message:

```text
giantComponent: Component "{{name}}" is {{lineCount}} lines — consider breaking it into smaller focused components
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-giant-component": "error"
	}
}
```
