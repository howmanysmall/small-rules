## Rule Details

This rule is configured as `small-rules/no-render-helper-functions` and reports the concrete syntax covered by its implementation in `src/rules/no-render-helper-functions.ts`.

## Configuration

<RuleOptions rule="no-render-helper-functions" />

## Examples

<RuleExample type="fail" title="Fails">

```tsx
function createLabel() {
	return <div />;
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
function Component() {
	return <div />;
}
```

</RuleExample>
