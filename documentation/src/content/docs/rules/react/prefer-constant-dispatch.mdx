## Rule Details

This rule is configured as `small-rules/prefer-constant-dispatch` and reports the concrete syntax covered by its implementation in `src/rules/prefer-constant-dispatch.ts`.

## Configuration

<RuleOptions rule="prefer-constant-dispatch" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
const type = "OPEN";

function Component() {
	const [state, dispatch] = useReducer(reducer, initialState);

	dispatch({ type });

	return state;
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function Component(type) {
	const [state, dispatch] = useReducer(reducer, initialState);

	dispatch({ type });

	return state;
}
```

</RuleExample>
