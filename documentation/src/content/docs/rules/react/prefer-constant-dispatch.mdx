`small-rules/prefer-constant-dispatch` reports code described by its source metadata: `disallow inline useReducer action objects that could be module-level constants`. It reports with the message `Move this inline useReducer action object to a module-level constant.`.

## Rule Details

This rule is configured as `small-rules/prefer-constant-dispatch` and reports the concrete syntax covered by its implementation in `src/rules/prefer-constant-dispatch.ts`.

<Aside type="note">

Rule message:

```text
preferConstantDispatch: Move this inline useReducer action object to a module-level constant.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/prefer-constant-dispatch": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
const type = "OPEN";

function Component() {
	const [state, dispatch] = useReducer(reducer, initialState);

	dispatch({ type });

	return state;
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function Component(type) {
	const [state, dispatch] = useReducer(reducer, initialState);

	dispatch({ type });

	return state;
}
```

</RuleExample>
