## Rule Details

This rule is configured as `small-rules/require-named-effect-functions` and reports the concrete syntax covered by its implementation in `src/rules/require-named-effect-functions.ts`.

## Configuration

## Examples

<RuleExample type="fail" title="Fails">

```ts
useEffect(() => {
	console.log("effect");
}, []);
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function handleEffect() {
	console.log("effect");
}
useEffect(handleEffect, []);
```

</RuleExample>
