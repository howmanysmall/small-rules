## Examples

<RuleExample type="fail" title="Fails">

```ts
const springs = useSpring({
	from: { x: 0 },
});
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
const springs = useSpring({
	from: { x: 0 },
	to: { x: 1 },
});
```

</RuleExample>
