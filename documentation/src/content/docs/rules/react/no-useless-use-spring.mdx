`small-rules/no-useless-use-spring` reports code described by its source metadata: `disallow useSpring hooks whose config and dependencies are entirely static`. It reports with the message `useSpring call has only static inputs and non-updating dependencies; replace it with a constant or remove the hook.`.

## Rule Details

This rule is configured as `small-rules/no-useless-use-spring` and reports the concrete syntax covered by its implementation in `src/rules/no-useless-use-spring.ts`.

<Aside type="note">

Rule message:

```text
uselessSpring: useSpring call has only static inputs and non-updating dependencies; replace it with a constant or remove the hook.
```

</Aside>

## Examples

<RuleExample type="fail" title="Fails">

```ts
const springs = useSpring({
	from: { x: 0 },
});
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
const springs = useSpring({
	from: { x: 0 },
	to: { x: 1 },
});
```

</RuleExample>
