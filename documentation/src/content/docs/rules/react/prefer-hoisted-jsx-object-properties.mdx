## Examples

<RuleExample type="fail" title="Fails">

```tsx
function View() {
	return <Component options={{ enabled: true, count: 5 }} />;
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
const PROPS = { enabled: true };

const view = <Component options={PROPS} />;
```

</RuleExample>
