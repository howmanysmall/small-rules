## Rule Details

This rule is configured as `small-rules/memoized-effect-dependencies` and reports the concrete syntax covered by its implementation in `src/rules/memoized-effect-dependencies.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```ts
import { useEffect } from "@rbxts/react";

function Component() {
	const dep = {};
	useEffect(() => {}, [dep]);
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
import { useEffect, useMemo, useCallback } from "@rbxts/react";

function Component() {
	const memo = useMemo(() => ({}), []);
	const callback = useCallback(() => {}, []);
	useEffect(() => {}, [memo, callback]);
}
```

</RuleExample>
