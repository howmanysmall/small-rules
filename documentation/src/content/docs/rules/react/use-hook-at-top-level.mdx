## Rule Details

## Examples

<RuleExample type="fail" title="Fails">

```ts
function Component() {
	if (condition) {
		useState(0);
	}
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
function Component() {
	useState(0);
}
```

</RuleExample>
