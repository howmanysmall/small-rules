## Rule Details

This rule is configured as `small-rules/require-react-component-keys` and reports the concrete syntax covered by its implementation in `src/rules/require-react-component-keys.ts`.

By default, the rule also reports root component `key` props because `allowRootKeys` is `false`.

## Examples

<RuleExample type="fail" title="Fails">

```tsx
function Bad1() {
	return (
		<>
			<div />
			<span />
		</>
	);
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
function Good1() {
	return <div />;
}
```

</RuleExample>
