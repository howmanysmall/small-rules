## Examples

<RuleExample type="fail" title="Fails">

```tsx
function Big({ a, b, c, d, e, f }) {
	return (
		<div>
			{a}
			{b}
			{c}
			{d}
			{e}
			{f}
		</div>
	);
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```tsx
function Small({ a, b }) {
	const [count, setCount] = useState(0);
	if (count > 0) setCount(count - 1);
	return (
		<div>
			<span>
				{a}
				{b}
			</span>
		</div>
	);
}
```

</RuleExample>
