## Rule Details

This rule is configured as `small-rules/no-unused-use-memo` and reports the concrete syntax covered by its implementation in `src/rules/no-unused-use-memo.ts`.

## Configuration

<RuleOptions rule="no-unused-use-memo" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
import { useMemo } from "@rbxts/react";

useMemo(() => 1, []);
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
import { useMemo } from "@rbxts/react";

const value = useMemo(() => 1, []);
```

</RuleExample>
