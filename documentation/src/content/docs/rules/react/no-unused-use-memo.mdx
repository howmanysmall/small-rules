`small-rules/no-unused-use-memo` reports code described by its source metadata: `disallow standalone useMemo calls that ignore the memoized value`. It reports with the message `useMemo results must be used. Standalone useMemo calls add overhead without preserving a value.`.

## Rule Details

This rule is configured as `small-rules/no-unused-use-memo` and reports the concrete syntax covered by its implementation in `src/rules/no-unused-use-memo.ts`.

<Aside type="note">

Rule message:

```text
unusedUseMemo: useMemo results must be used. Standalone useMemo calls add overhead without preserving a value.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-unused-use-memo": [
			"error",
			{
				"environment": "roblox-ts"
			}
		]
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
import { useMemo } from "react";

useMemo(() => 1, []);
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
import { useMemo } from "react";

const value = useMemo(() => 1, []);
```

</RuleExample>
