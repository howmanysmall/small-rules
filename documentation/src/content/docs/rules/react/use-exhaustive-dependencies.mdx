## Examples

<RuleExample type="fail" title="Fails">

```ts
function Component() {
	const [count, setCount] = useState(0);
	useEffect(() => {
		console.log(count);
	}, []);
}
```

</RuleExample>

When auto-fixed, this becomes:

```ts
function Component() {
	const [count, setCount] = useState(0);
	useEffect(() => {
		console.log(count);
	}, [count]);
}
```

<RuleExample type="pass" title="Passes">

```ts
function Component() {
	const a = 1 as any;
	const b = 2 as const satisfies number;
	useEffect(() => {
		console.log(a, b);
	}, [a, b]);
}
```

</RuleExample>
