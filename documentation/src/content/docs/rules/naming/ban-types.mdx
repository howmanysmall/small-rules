## Rule Details

This rule is configured as `small-rules/ban-types` and reports the concrete syntax covered by its implementation in `src/rules/ban-types.ts`.

<RuleOptions rule="ban-types" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
type User = {
	readonly name: string;
	readonly password: string;
};

type HiddenUser = Omit<User, "password">;
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
type User = {
	readonly name: string;
};

type ActiveUser = Pick<User, "name">;
```

</RuleExample>
