## Examples

<RuleExample type="fail" title="Fails">

```ts
enum SORTORDER {
	MostRecent,
	LeastRecent,
	Newest,
	Oldest,
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
enum SortOrder {
	MostRecent,
	LeastRecent,
	Newest,
	Oldest,
}
```

</RuleExample>
