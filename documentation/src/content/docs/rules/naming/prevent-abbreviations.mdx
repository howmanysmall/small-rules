## Rule Details

This rule is configured as `small-rules/prevent-abbreviations` and reports the concrete syntax covered by its implementation in `src/rules/prevent-abbreviations.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```ts
const err = new Error();
```

</RuleExample>

When auto-fixed, this becomes:

```ts
const error = new Error();
```

<RuleExample type="pass" title="Passes">

```ts
const ERR = "error";
```

</RuleExample>
