`small-rules/prefer-singular-enums` reports code described by its source metadata: `prefer singular naming for enums`. It reports with the message `Enum name "{{name}}" should be singular.`.

## Rule Details

This rule is configured as `small-rules/prefer-singular-enums` and reports the concrete syntax covered by its implementation in `src/rules/prefer-singular-enums.ts`.

<Aside type="note">

Rule message:

```text
notSingular: Enum name "{{name}}" should be singular.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/prefer-singular-enums": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
enum SortOrders {
	MostRecent,
	LeastRecent,
	Newest,
	Oldest,
}
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
enum SortOrder {
	MostRecent,
	LeastRecent,
	Newest,
	Oldest,
}
```

</RuleExample>
