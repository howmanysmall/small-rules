`small-rules/prefer-idiv` reports code described by its source metadata: `prefer .idiv() for integer division instead of math.floor(x / y)`. It reports with the message `Use .idiv() instead of math.floor(x / y) for integer division.`.

## Rule Details

This rule is configured as `small-rules/prefer-idiv` and reports the concrete syntax covered by its implementation in `src/rules/prefer-idiv.ts`.

<Aside type="note">

Rule message:

```text
useIdiv: Use .idiv() instead of math.floor(x / y) for integer division.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/prefer-idiv": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
math.floor(x / y);
```

</RuleExample>

When auto-fixed, this becomes:

```ts
x.idiv(y);
```

<RuleExample type="pass" title="Passes">

```ts
x.idiv(y);
```

</RuleExample>
