## Rule Details

This rule is configured as `small-rules/prefer-idiv` and reports the concrete syntax covered by its implementation in `src/rules/prefer-idiv.ts`.

## Configuration

<RuleOptions rule="prefer-idiv" />

## Examples

<RuleExample type="fail" title="Fails">

```ts
math.floor(x / y);
```

</RuleExample>

When auto-fixed, this becomes:

```ts
x.idiv(y);
```

<RuleExample type="pass" title="Passes">

```ts
x.idiv(y);
```

</RuleExample>
