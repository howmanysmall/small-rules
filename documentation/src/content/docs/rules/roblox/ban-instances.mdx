`small-rules/ban-instances` reports code described by its source metadata: `ban specified Roblox Instance classes in new Instance() calls and JSX elements`. It reports with the message `Instance class '{{className}}' is banned by project configuration. This class may cause performance issues, is deprecated, or has a better alternative. Check project guidelines for the recommended replacement.`.

## Rule Details

This rule is configured as `small-rules/ban-instances` and reports the concrete syntax covered by its implementation in `src/rules/ban-instances.ts`.

<Aside type="note">

Rule messages:

```text
bannedInstance: Instance class '{{className}}' is banned by project configuration. This class may cause performance issues, is deprecated, or has a better alternative. Check project guidelines for the recommended replacement.
bannedInstanceCustom: {{customMessage}}
bannedProperty: Property '{{propertyName}}' on Instance class '{{className}}' is banned by project configuration. This property may cause performance issues, is deprecated, or has a better alternative. Check project guidelines for the recommended replacement.
bannedPropertyCustom: {{customMessage}}
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/ban-instances": ["error", {}]
	}
}
```
