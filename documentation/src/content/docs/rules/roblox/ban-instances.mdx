## Rule Details

This rule is configured as `small-rules/ban-instances` and reports the concrete syntax covered by its implementation in `src/rules/ban-instances.ts`.

It can ban Instance classes and configured properties on JSX elements or imperative assignments.

## Examples

<RuleExample type="fail" title="Fails">

```ts
new Instance("Part");
```

```tsx
const view = <uisizeconstraint MaxSize={new Vector2(100, 100)} />;
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
new Instance("Folder");
```

</RuleExample>
