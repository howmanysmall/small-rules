`small-rules/no-array-size-assignment` reports code described by its source metadata: `disallow array append assignments using array[array.size()] = value and prefer push-based appends`. It reports with the message `Do not append with array[array.size()] = value. Use array.push(value) instead.`.

## Rule Details

This rule is configured as `small-rules/no-array-size-assignment` and reports the concrete syntax covered by its implementation in `src/rules/no-array-size-assignment.ts`.

<Aside type="note">

Rule message:

```text
usePush: Do not append with array[array.size()] = value. Use array.push(value) instead.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-array-size-assignment": [
			"error",
			{
				"allowAutofix": false
			}
		]
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
array[array.size()] = value;
```

</RuleExample>

When auto-fixed, this becomes:

```ts
array.push(value);
```

<RuleExample type="pass" title="Passes">

```ts
array.push(value);
```

</RuleExample>
