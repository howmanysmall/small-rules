## Rule Details

This rule is configured as `small-rules/no-useless-default` and reports the concrete syntax covered by its implementation in `src/rules/no-useless-default.ts`.

This rule is Roblox-specific: it compares JSX intrinsic properties against known Roblox Instance defaults.

## Configuration

<RuleOptions rule="no-useless-default" />

## Examples

<RuleExample type="fail" title="Fails">

```tsx
const view = <uiaspectratioconstraint AspectRatio={1} />;
```

</RuleExample>

When auto-fixed, this becomes:

```tsx
const view = <uiaspectratioconstraint />;
```

<RuleExample type="pass" title="Passes">

```tsx
const view = <uiaspectratioconstraint AspectRatio={2} />;
```

</RuleExample>
