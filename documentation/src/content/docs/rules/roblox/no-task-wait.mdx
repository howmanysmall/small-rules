`small-rules/no-task-wait` reports code described by its source metadata: `disallow task.wait() calls`. It reports with the message `Do not use Promise.delay(...).await() in tests. Advance deterministic schedulers or wait for the actual condition instead.`.

## Rule Details

This rule is configured as `small-rules/no-task-wait` and reports the concrete syntax covered by its implementation in `src/rules/no-task-wait.ts`.

<Aside type="note">

Rule messages:

```text
noPromiseDelayAwait: Do not use Promise.delay(...).await() in tests. Advance deterministic schedulers or wait for the actual condition instead.
noTaskWait: Do not use task.wait() in tests. Advance deterministic schedulers or wait for the actual condition instead.
```

</Aside>

## Configuration

```json
{
	"plugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-task-wait": "error"
	}
}
```

## Examples

<RuleExample type="fail" title="Fails">

```ts
task.wait();
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
callback();
```

</RuleExample>
