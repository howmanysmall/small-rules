## Rule Details

This rule is configured as `small-rules/no-task-wait` and reports the concrete syntax covered by its implementation in `src/rules/no-task-wait.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```ts
task.wait();
```

```ts
Promise.delay(0).await();
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
callback();
```

</RuleExample>
