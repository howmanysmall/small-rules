## Rule Details

This rule is configured as `small-rules/enforce-ianitor-check-type` and reports the concrete syntax covered by its implementation in `src/rules/enforce-ianitor-check-type.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```ts
const isUser = Ianitor.strictInterface({
	name: Ianitor.string,
	age: Ianitor.number,
	profile: Ianitor.interface({
		email: Ianitor.string,
		settings: Ianitor.record(Ianitor.string, Ianitor.unknown),
	}),
});
```

</RuleExample>

<RuleExample type="pass" title="Passes">

```ts
type Simple = string;
```

</RuleExample>
