## Rule Details

This rule is configured as `small-rules/prefer-udim2-shorthand` and reports the concrete syntax covered by its implementation in `src/rules/prefer-udim2-shorthand.ts`.

## Examples

<RuleExample type="fail" title="Fails">

```ts
new UDim2(1, 0, 1, 0);
```

</RuleExample>

When auto-fixed, this becomes:

```ts
UDim2.fromScale(1, 1);
```

<RuleExample type="pass" title="Passes">

```ts
new UDim2(1, 2, 3, 4);
```

</RuleExample>
