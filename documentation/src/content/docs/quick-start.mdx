## Install

Add `small-rules` and `oxlint` to your roblox-ts project:

## Configure

Create or update your `.oxlintrc.json` to register the plugin and enable rules:

```json
{
	"jsPlugins": ["@pobammer-ts/small-rules"],
	"rules": {
		"small-rules/no-print": "error",
		"small-rules/no-warn": "error",
		"small-rules/ban-react-fc": "error",
		"small-rules/no-task-wait": "error",
		"small-rules/prefer-early-return": "warn"
	}
}
```

<Aside type="note">
	Start with a few high-impact rules and gradually add more. The `error` severity will fail the lint check, while
	`warn` will show a warning without failing.
</Aside>

## Run

Run the linter manually:

```bash
oxlint .
```

Or add it to your `package.json` scripts:

```json
{
	"scripts": {
		"lint": "oxlint ."
	}
}
```

Then run:

```bash
aube run lint
```

## Auto-fix

Many rules provide automatic fixes. Apply them with:

```bash
oxlint --fix .
```

<Aside type="caution">
	Always review auto-fixes before committing. While the fixes are designed to be safe, edge cases can occur.
</Aside>
